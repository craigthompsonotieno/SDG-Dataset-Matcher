# SDG Dataset Matcher

A Python project developed as part of a UN-DESA initiative to match datasets with relevant Sustainable Development Goals (SDGs).

## Objectives

- Identify relevant SDGs for datasets
- Improve dataset discoverability
- Support SDG-aligned data cataloguing

## Tools Used

- Python
- Pandas
- OpenPyXL
- Jupyter Notebook
